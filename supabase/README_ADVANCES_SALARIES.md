# Установка функционала Авансов и Зарплат

## Шаг 1: Выполнить SQL миграцию

Выполните SQL скрипт в Supabase SQL Editor:

```sql
-- Файл: supabase/migrations/030_add_advances_and_salaries.sql
```

Это создаст таблицы:
- `events` - ивенты для авансов
- `advances` - авансы
- `employees` - сотрудники для зарплат
- `salaries` - зарплаты

## Шаг 2: Установить библиотеки

Выполните в терминале:

```bash
npm install jspdf jspdf-autotable
```

## Шаг 3: Готово!

После выполнения миграции и установки библиотек функционал будет доступен на вкладках "Авансы" и "Зарплаты" в панели главного организатора.

### Функционал Авансов:
- Вкладки с ивентами (можно создавать, переименовывать, удалять)
- Таблица с полями: Дата, Сумма, Назначение, Способ оплаты
- Экспорт в PDF

### Функционал Зарплат:
- Вкладки с сотрудниками (можно создавать, переименовывать, удалять)
- Таблица с полями: Месяц, Зарплата, Бонус, Координация, Общая
- Автоматический расчет общей суммы
- Экспорт в PDF

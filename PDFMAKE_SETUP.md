# Настройка pdfmake для поддержки кириллицы

## Что сделано

Компоненты `AdvancesTab` и `SalariesTab` переписаны для использования `pdfmake` вместо `jsPDF`.

## Установка

Выполните в терминале:
```bash
npm install pdfmake
```

## Преимущества

- ✅ Полная поддержка кириллицы из коробки
- ✅ Не нужно конвертировать шрифты
- ✅ Встроенный шрифт Roboto с поддержкой кириллицы
- ✅ Простой и понятный API

## Что изменилось

1. **Импорты:**
   - Заменены `jsPDF` и `jspdf-autotable` на `pdfmake`
   - Добавлена инициализация шрифтов

2. **Генерация PDF:**
   - Используется `pdfMake.createPdf()` вместо `new jsPDF()`
   - Таблицы создаются через объект `docDefinition`
   - Кириллица работает автоматически

## Готово!

После установки `pdfmake` кириллица будет корректно отображаться в PDF без дополнительных настроек.
